Toys API Documentation


Users

URL Name URL_Input Body_Input Description
POST /users register None {
"name": "John Doe",
"email": "john2@gmail.com",
"password": "pass1234",
"passwordConfirm": "pass1234"
}		 Adds a user
POST /users/login login None {
"email": "john11@gmail.com",
"password": "pass1234"
}		 Searchs for toys by their name or info

Toys

URL Name URL_Input Body_Input Description
GET /toys getToys None None Gets all the toys
GET /toys/search?s=value toySearch s=name
s=info		 None Searchs for toys by their name or info
GET /toys/category/:catnameValue toyCategory catname None Gets toys by category
GET /toys/prices/?min=minValue&max=maxValue getByPrice min=minValue&max=maxValue None Gets toys by price range
GET /toys/single/:idValue singleById id None Gets a toy by its id
POST /toys addNewToy None {
"name": "harry potter lego set",
"info": "harry potter lego set for ages 8-99",
"category": "lego",
"img_url": "https://cdn.pixabay.com/photo/2012/04/16/13/25/fireman-35994_1280.png",
"price": 45
}		 Adds a new toy to a user
PUT /toys/:id editToy None {
"price": 100
}		 Updates a toy
DELETE /toys/:id deleteToy id None Deletes a toy
In the gets can add ?page=numOfPage to pick a page
